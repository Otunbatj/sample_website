Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
ol ordered list making an ordered list
ul unordered list making an unordered list
li list making an list, needs to be nested inside ol or ul
blockquote block quotation show formatted quotation
br break Enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make important text <strong>technical sophistication</strong> technical sophistication
a anchor tag insert hyperlinks <a href="learnenough.com">link</a> link
img image tag insert image <img src="/images/kitten.jpg" alt="An adorable kitten">
code code tag format a text as code <code>formatted code</code> formatted code
span span tag define inline section in document <span>Call me Ishmael</span> Call me Ishmael

Document Tags

Tag Name Purpose
html html tag They encapsulate the whole html page
head head tag This is used to provide header information for the page including page title and metadata
body body tag This is used to house the body of the page, this is the part of the html page that is visible
title title tag This is used to provide the title to that is displayed on the browser tab
meta meta tag This is used to provide extra information to the page e.g. encoding format
